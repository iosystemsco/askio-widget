‚öôÔ∏è Available Options

Position Options:

  • bottom-right - Bottom right corner (default)
  • bottom-left - Bottom left corner
  • bottom-center - Bottom center
  • top-right - Top right corner
  • top-left - Top left corner
  • top-center - Top center
  • left-center - Left side, vertically centered
  • right-center - Right side, vertically centered
  • inline - Embedded inline (no floating button)

Dimension Options:

  • width - Widget width (e.g., "384px", "50%", "20rem")
  • height - Widget height (e.g., "600px", "80vh")
  • maxWidth - Maximum width
  • maxHeight - Maximum height
  • minWidth - Minimum width
  • minHeight - Minimum height

Other Options:

  • hideButton - Hide the floating action button (useful for custom triggers)
Note: This is a demo page. Replace your-site-token with your actual site token to test with real data.
